SeanUI

SeanUI is a bunch of CSS that looks nicer than not using it. Keep in mind that in most cases, it is not the same CSS as used in this webpage. SeanUI is still a work in progress. Consider joining the Eskroem Fanclub to get notified when it’s done.

If you’re directly copying stuff, use these variables

/* For fonts, @import Roboto Flex and JetBrains Mono
   from https://fonts.google.com */

:root {
  --text: "Roboto Flex", sans-serif;
  --code: "JetBrains Mono", monospace;
  --bg0: #000;
  --bg-color: #151515;
  --bg2-color: #2a2a2a;
  --bg3-color: #3f3f3f;
  --txt-color: #fff;
  --red: #f55;
  --blue: #55f;
  --purple: #f5f;
}

body {
  font-family: var(--text);
  background-color: var(--bg-color);
  color: var(--txt-color);
  text-align: justify;
  transition: 0.2s;
}

<a> Anchor

Anchor Anchor, Link Anchor, Visited 
a {
  color: var(--red);
  text-decoration: none;
}

a:link {
  color: var(--blue);
  text-decoration: none;
}

a:visited {
  color: var(--purple);
  text-decoration: none;
}

a:hover {
  text-decoration: underline solid currentColor 1px;
}

<h1>–<h6> Section Heading 1–6

You can replace h1 with h1–h6. If you don’t like the bottom border, remove border-bottom.

h1 /* With Border */ {
  font-size: 32px;
  font-weight: 300;
  border-bottom: 1px solid var(--bg2-color);
  margin: 16px;
}
h1 /* Without Border */ {
  font-size: 32px;
  font-weight: 300;
  margin: 16px;
}
